# Звіт: Перевірка Anti-Fraud системи Pixalate

**Дата:** 06.03.2026
**Сервіс:** [Pixalate Fraud API](https://www.pixalate.com/) (v2)
**Мета:** Перевірити, чи CTV-емулятор проходить fraud-скорінг Pixalate без детекції
**Результат:** Пройдено 40/40 тестів

---

## 1. Що таке Pixalate

Pixalate — провідний сервіс виявлення рекламного фроду, який використовується DSP та SSP платформами для фільтрації невалідного трафіку (IVT). Сервіс аналізує IP-адресу, User-Agent та інші сигнали і повертає **fraud probability** — число від 0 до 1:

| Probability | Інтерпретація |
|---|---|
| 0.00 – 0.25 | Легітимний трафік |
| 0.25 – 0.75 | Підозрілий трафік |
| 0.75 – 1.00 | Фродовий трафік |

Наш поріг: **0.25** — все що нижче вважається чистим.

---

## 2. Методологія тестування

Тест складається з 4 секцій:

1. **API connectivity** — валідація ключа та доступності сервісу
2. **Fraud scoring** — реальні запити до Pixalate з різними профілями (контрольний + 3 CTV)
3. **Fingerprint audit** — перевірка що browser fingerprint відповідає реальному Smart TV
4. **Chromium anti-detection** — перевірка приховання автоматизації браузера

### Контрольний тест

Для валідації що Pixalate дійсно працює, спершу надсилається запит з реальними даними сервера (IP дата-центру + HeadlessChrome UA). Очікуваний результат — детекція як фрод.

### CTV-профілі

Генеруються 3 профілі під різні Smart TV платформи:
- **AndroidTV** — NVIDIA SHIELD, TCL, Xiaomi Mi Box
- **Tizen** — Samsung Smart TV
- **WebOS** — LG Smart TV

Кожен профіль включає: residential IP, реалістичний TV User-Agent, hardware fingerprint.
